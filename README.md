# B+W Engineering Status Page

Infrastructure status page for B+W Engineering web services.

## Services Monitored
- bwengr.com - Main website and portfolio
- drainageplans.la - Altadena fire rebuild drainage plan landing page
- gradingplans.la - Los Angeles grading plans landing page
- civilengineer.la - Civil engineer LA redirect service

## Tech Stack
- Astro v6
- Cloudflare Pages
- Tailwind CSS v4

## Development

```bash
npm install
npm run dev
```

## Deploy
Connected to Cloudflare Pages via GitHub. Push to main branch triggers automatic deployment to status.bwengr.com.

## SEO
- WebPage schema with Organization
- Sitemap auto-generated
- Meta description for search engines
- Internal link to bwengr.com in footer
